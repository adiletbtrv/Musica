## Features

- **Global Discovery**: Real-time integration with the iTunes Search API to fetch charts, new releases, and global search results.
- **Advanced Audio Engine**: Custom-built media player featuring track seeking, volume normalization, shuffle, and intelligent repeat modes.
- **Library Persistence**: Local-first architecture that saves your "Liked" songs and custom collections directly to the browser storage.
- **Playlist Studio**: Create and manage personal playlists with custom theme colors and a dedicated drag-and-drop edit interface.
- **Mobile-First UX**: 
  - Adaptive sidebar for desktop and intuitive bottom navigation for mobile.
  - Expandable full-screen mobile player.
  - Context-aware menus (right-click/long-press) for rapid library management.
- **Tech Stack**: Vanilla JavaScript (ES6+), CSS3 (Variables & Grid), HTML5, iTunes API.

1.  Clone the repository.

    ```bash
    npm install
    ```

    ```bash
    npm run dev
    ```
